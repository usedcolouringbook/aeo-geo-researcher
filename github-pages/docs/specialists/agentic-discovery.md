# Agentic Discovery

**Research domain:** MCP servers, agent pipelines, agent-native formats, and agentic marketing — the practice of getting AI agents to take actions for customers, not just find content.

## When the Orchestrator Routes Here

- The client's question is about AI agents taking actions, not just finding content
- The client wants to understand MCP or agent-native integrations
- The client is thinking about being discovered inside agentic workflows

## What This Specialist Researches

| Area | What it covers |
|---|---|
| MCP servers | Model Context Protocol — how agents access tools and data sources |
| Agent discovery | How agents find and select tools/services |
| Agent-native formats | What data formats agents prefer when taking actions |
| Agentic marketing | Getting brands into agent workflows as a trusted action target |

## The Distinction

AEO/GEO is about being found in AI answers (retrieval, citations, SOM).
Agentic marketing is about being chosen for AI-driven actions (booking, purchasing, routing).

This specialist covers the agentic layer. The other four specialists cover the retrieval and citation layer.
